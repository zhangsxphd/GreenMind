# Smart Agri

设施农业智能决策平台，全栈前后端分离项目。

## 目录结构

- `frontend`: Vite + React 18 + React Router + Tailwind CSS 前端应用
- `backend`: Fastify 后端服务骨架

## 启动方式

1. 安装依赖：`source ~/.nvm/nvm.sh && npm install`
2. 启动前端：`source ~/.nvm/nvm.sh && npm run dev:frontend`
3. 启动后端：`source ~/.nvm/nvm.sh && npm run dev:backend`
